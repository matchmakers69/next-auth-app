#### Semantic HTML.

**Using HTML landmark elements for structural grouping improves accessibility and semantic clarity.**

1. - **Better Accessibility** - Screen readers can navigate the page more easily.
2. - **Semantic Clarity** - Improves the readability of the code.

```html
<main>
  <header>
    <nav>
      <ul>
        <li><a href="/">Home</a></li>
        <li><a href="/about">About</a></li>
        <li><a href="/contact">Contact</a></li>
      </ul>
    </nav>
  </header>
  <section>
    <h1>Accessibility</h1>
    <p>
      Web accessibility (also referred to as a11y) is the design and creation of
      websites that can be used by everyone.
    </p>
  </section>
  <footer>
    <p>&copy; 2025</p>
  </footer>
</main>
```
